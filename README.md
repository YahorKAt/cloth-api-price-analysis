# 📊 ClothAPI Price Analysis

Асинхронный JavaScript-проект для анализа цен товаров из **ClothAPI**. Скрипт рассчитывает среднюю цену самых дешевых 10% товаров (10-й перцентиль) в каждой категории.

В репозитории также содержится **Postman-коллекция** для тестирования и отладки API-запросов.

## 🚀 Особенности

- ✅ **Асинхронная работа**: Использование `async/await` и `fetch` для неблокирующих запросов.
- ✅ **Пагинация**: Корректная обработка многостраничных ответов API (`has_next`, `total_count`).
- ✅ **Оптимизация**: Загрузка только необходимого количества товаров (не более 10% от категории), а не всей выборки.
- ✅ **Обработка ошибок**: Информативные сообщения об HTTP-ошибках (4xx, 5xx, network errors).
- ✅ **Структурированный вывод**: Результаты в виде `Map<CategoryName, AvgPrice>`.
- ✅ **Postman-коллекция**: Готовые запросы для ручной проверки эндпоинтов.

## 🛠 Технологии

| Технология | Версия | Назначение |
|------------|--------|------------|
| JavaScript | ES2020+ | Основная логика |
